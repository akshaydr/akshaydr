Ajit, A Social and Intelligent Humanoid Robot

The project was conducted in 2018-19 at KLE Technological University, Hubli, India. The main objective of this project is to assist in the social inclusion of elderly person and young person with disabilities.

It is a 24 DOF humanoid robot. The robot contains Intel NUC as the main controller and an Intel Movidiuous for fast neural processing. The head has Intel Realsense camera for object analytics and OLED display for expressing emotions. Microphone is attached for speech recognition and speakers are present for talking. Each hand has 5 DOF which can be used for gestures and also pick and place small objects. Dynamixel actuators are used. Robot is capable of both legged locomotion and wheeled locomotion. Detachable wheel base is to be developed. Robot is modelled using SolidWorks. All joints are 3D printed using PLA material. Carbon fiber pipes are used in the backbone of the robot for greater strength.


The project was presented at an exhibit "Engineering Education Festa" in Ilsam, South Korea. We were the only team from India to participate in this exhibit. Few snapshots from the exhibit is shown below.
